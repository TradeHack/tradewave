## Description

[Nextjs](https://github.com/vercel/next.js) TypeScript Tradewave frontend repo.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development - running on port 3000
$ npm run dev

```

## Test

```bash
# unit tests
$ npm run test
```

## Dependencies

This repo needs the backend-api repo running
